# Дизайн-система «Маяк»

UI-кит для продукта о спокойном, понятном и персональном старте в новой компании.
Интерфейс — это **умный цифровой проводник**, а не HR-система, LMS или корпоративный дашборд.

> Все токены определены в `app/globals.css` через `@theme inline` и `:root`.
> Никогда не используйте «сырые» цвета (`#fff`, `text-white`, `bg-black`) — только токены.

---

## 1. Философия

| Транслируем | Избегаем |
|---|---|
| Ясность, навигация, движение вперёд | Контроль, экзамен, бюрократия |
| Поддержка, доверие, спокойствие | Слежка за сотрудником, тревожность |
| Персонализация, интеллект | Шаблонный SaaS, корпоративная админка |
| Мягкая технологичность, воздух | Холодность, кислотные цвета, агрессивная геймификация |

Образы продукта: **карта пути**, **созвездие знаний**, **умный ассистент**, **персональная навигационная система**.

Каждый экран должен отвечать на три вопроса новичка: «Где я сейчас?», «Что дальше?», «Кто поможет?».

## 2. Цвет

Палитра ограничена. Не вводите новые цвета без необходимости — расширяйте через `color-mix`.

### Основные роли

| Токен | Назначение | Tailwind-класс |
|---|---|---|
| `--background` | Тёплый воздушный фон-«бумага» | `bg-background` |
| `--foreground` | Основной текст | `text-foreground` |
| `--card` / `--card-foreground` | Карточки, панели | `bg-card text-card-foreground` |
| `--primary` | Спокойный teal — рост, доверие, спокойствие | `bg-primary text-primary-foreground` |
| `--secondary` | Тихие подложки, второстепенные кнопки | `bg-secondary text-secondary-foreground` |
| `--muted` / `--muted-foreground` | Приглушённые поверхности и подписи | `bg-muted text-muted-foreground` |
| `--accent` | Тёплый песочный акцент — дружелюбие | `bg-accent text-accent-foreground` |
| `--border` / `--input` | Границы и поля ввода | `border-border` |
| `--ring` | Фокус-кольцо (teal) | `ring-ring` |

### Поверхности и атмосфера

| Токен | Назначение |
|---|---|
| `--deep`, `--deep-foreground`, `--deep-muted`, `--deep-border` | Глубокие teal-панели для иммерсивных блоков (hero, summary). Спокойные, **никогда не резкие** |
| `--aurora-1/2/3` | Мягкое ambient-свечение фона (teal → мята → песок) |
| `--shadow-soft`, `--shadow-card` | Мягкие тонированные тени (тень primary, не серая) |

### Правила цвета

- **3–5 цветов максимум** на экране: primary teal + 2–3 нейтрали + песочный акцент.
- **Никакого фиолетового/violet** и кислотных оттенков.
- Поменяли фон элемента — **обязательно** поменяйте и цвет текста (контраст).
- Градиенты — только как тонкий ambient-акцент (aurora), аналоговые цвета, 2–3 стопа. Никогда на основных элементах.

## 3. Типографика

Две гарнитуры, обе подключены в `app/layout.tsx`:

| Шрифт | Переменная | Класс | Назначение |
|---|---|---|---|
| Geist | `--font-geist-sans` | `font-sans` | Заголовки и весь текст |
| Geist Mono | `--font-geist-mono` | `font-mono` | Кикеры, метки, числа, технические подписи |

Правила:
- Межстрочный для body: `leading-relaxed` / `leading-6` (1.4–1.6).
- Заголовки и важные строки — `text-balance` / `text-pretty`.
- Минимум 14px для body. Декоративные шрифты для текста — нельзя.
- Кикеры/метки: `font-mono`, uppercase, `tracking-wide`, цвет `text-muted-foreground` или `text-accent-foreground`.

## 4. Форма и пространство

| Свойство | Значение | Заметки |
|---|---|---|
| Базовый радиус | `--radius: 1.25rem` | `rounded-lg` = базовый, `rounded-xl` крупнее, `rounded-md/sm` мельче |
| Кнопки / чипы | пилл-форма (`rounded-full`) | мягкие, дружелюбные |
| Тени | `var(--shadow-card)` для карточек, `var(--shadow-soft)` для приподнятых блоков | тонированы в primary, не серые |
| Отступы | шкала Tailwind (`p-4`, `gap-6`, `py-8`) | без произвольных `p-[17px]` |
| Раскладка | flexbox по умолчанию, grid только для 2D | без float/absolute без необходимости |
| Spacing | только `gap-*` между элементами | не смешивать margin/padding с gap; не использовать `space-*` |

Дизайн **mobile-first** — сначала мобайл, затем `md:` / `lg:` улучшения.

## 5. Анимации

Доступны утилиты (определены в `globals.css`). Движение мягкое, медленное, «дышащее» — без резкости.

| Класс | Эффект | Применение |
|---|---|---|
| `.animate-aurora` | Дрейф ambient-свечения (16s) | фоновые пятна |
| `.animate-float` | Лёгкое плавание (6s) | акцентные иконки, узлы |
| `.animate-pulse-ring` | Пульс-кольцо teal (2.6s) | активный шаг, «сейчас» |
| `.twinkle` (keyframes) | Мерцание | звёзды созвездия знаний |

## 6. Паттерны компонентов

Эталонные реализации — в `src/modules/personal-space/ui/`.

- **Ambient-фон** (`AmbientBackground.tsx`) — слой aurora-пятен под контентом, `pointer-events-none`.
- **Карта пути** (`JourneyMap.tsx`) — таймлайн со статусами `done / current / upcoming`; активный шаг с `animate-pulse-ring`.
- **Созвездие знаний** (`KnowledgeConstellation.tsx`) — SVG-узлы и связи вместо таблиц.
- **Умный проводник** (`Assistant.tsx`) — карточка ассистента, чипы-подсказки, мягкие пузыри диалога.
- **Фокус на сегодня** (`TodayFocus.tsx`) — интерактивные задачи, не «отчётность ради отчётности».
- **Глубокая панель** — фон `var(--deep)`, текст `var(--deep-foreground)` для иммерсивных hero/summary блоков.

### Статусы (единый язык)
- `done` — teal-заливка/галочка, спокойно.
- `current` — пульс-кольцо, акцент внимания.
- `upcoming` — приглушённо (`muted`), без тревоги.

## 7. Чек-лист перед сдачей экрана

- [ ] Только токены, нет «сырых» цветов и `text-white`/`bg-black`.
- [ ] 3–5 цветов, нет фиолетового и кислотных.
- [ ] Поменян фон → поменян текст (контраст ≥ AA).
- [ ] Пилл-кнопки, скруглённые карточки, мягкие тонированные тени.
- [ ] `font-mono` для кикеров/чисел, `font-sans` для остального.
- [ ] Mobile-first, проверено на узком вьюпорте.
- [ ] flexbox/`gap-*`, без `space-*` и произвольных пикселей.
- [ ] Нет тяжёлых таблиц, HR-вайба и ощущения слежки.
- [ ] Экран отвечает: «Где я? Что дальше? Кто поможет?».
- [ ] Иконки из `lucide-react`, не эмодзи; alt у изображений.
